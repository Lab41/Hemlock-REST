Hemlock-REST

Hemlock REST Server. Same functionality as the Hemlock CLI, but through a RESTful interface.

Install instructions

Required dependencies

Hemlock

Python modules:

  • web

Running

python rest-server.py 80

Alternatively, rest-server.py can be modified to support more robust webservers.
